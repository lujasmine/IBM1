Effortless and Secure Banking

Systems Engineering Project

Introduction

This website contains all the documentation for the "Effortless and Secure Banking" project for IBM as part of the UCL Computer Science Systems Engineering Module for 2nd year students.

The aim of our project was to create an innovative and new banking solution that is effortless but does not compromise security for the user.

We have created a login authentication system that incorporates a web app, iOS app and a predictive engine.

Product Features



Native iOS App

Native mobile app is available on iOS devices (iOS 6+). Users with the iOS app can login more conveniently with TouchID or Prediction Engine.

Web App

The web app provides various login options, without having to enter a password. Can be used in conjunction with the mobile app for simple login.

Prediction Engine

A fine tuned Machine Learning algorithm has been implemented into the native iOS app for a hands-free process of logging in. The prediction engine takes into account 4 factors and analyzes the user's login patterns before allowing the user to successfully login.
